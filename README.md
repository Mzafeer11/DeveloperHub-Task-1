# DeveloperHub Task 1: Iris Analysis

This repository contains the solution for Task 1 of the DeveloperHub Internship program. 

## Project Overview
The project focuses on analyzing the classic Iris dataset. It includes data exploration, preprocessing, and building a machine learning model to classify iris flower species based on their sepal and petal dimensions.

## Files
- `iris-analysis.ipynb`: The main Jupyter Notebook containing all the code, from data loading and visualization to model training and evaluation.

## How to Run
1. Ensure you have Python installed along with Jupyter Notebook or JupyterLab.
2. Install the necessary libraries (e.g., pandas, scikit-learn, matplotlib, seaborn).
3. Open `iris-analysis.ipynb` and run the cells sequentially.
